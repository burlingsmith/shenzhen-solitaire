# Shenzhen Solitaire
Mockup of Zachtronics' _SHENZHEN SOLITAIRE_.

Last updated for Elm 0.19.

## Controls
- Cards are moved by clicking around.
- `ESC` and `SPACE` pause/unpause the game.
- `R` starts a new game.
